# E-commerce Delivery Optimization Model

## 📌 Overview
This project focuses on optimizing delivery routes and reducing delivery time using Python and Streamlit.

## 🚀 Features
- Optimized route selection
- Reduced delivery time
- Interactive user interface

## 🛠️ Tech Stack
- Python
- Streamlit

## ▶️ How to Run
1. Install required libraries
2. Run the command:
   streamlit run app.py

## 📊 Use Case
Helps e-commerce companies improve delivery efficiency and reduce costs.
